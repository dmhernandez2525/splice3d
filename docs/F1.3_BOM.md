# F1.3 Bill of Materials

## Scope

This feature provides a complete machine-readable BOM with cost tiers, sourcing redundancy, donor-part mapping, build difficulty ratings, and shipping estimates.

## Artifacts

1. Catalog data: `hardware/f1_3/spec/bom_catalog.json`
2. Validation module: `postprocessor/bom_validation.py`
3. Validation script: `scripts/hardware/validate_f1_3.py`
4. Tests: `postprocessor/tests/test_bom_validation.py`
