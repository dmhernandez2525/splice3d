# cable_management

## Print Settings

1. Material: PETG
2. Layer height: 0.20mm
3. Infill: 25%
4. Perimeters: 4
5. Orientation: clip-side-up
6. Supports: off for critical functional areas

## Notes

1. Use 0.4mm nozzle.
2. Keep part cooling enabled for bridge stability.
3. Validate hole clearances with a 3.0mm and 6.0mm gauge after print.
